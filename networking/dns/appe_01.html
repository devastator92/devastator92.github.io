E.1. BIND Name Server Boot File Directives and Configuration File Statements

Here's a handy list of all the boot file directives and configuration file statements for the BIND name server, as well as configuration directives for the BIND resolver. Some of the directives and statements exist only in later versions, so your name server may not support them yet. Newer directives or statements are labeled with the specific version of BIND in which they were introduced (e.g., 8.2+). If they've been around a long time, they aren't labeled.
