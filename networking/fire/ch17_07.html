17.7. Related Protocols

Under Unix, rcp is also frequently used for file transfer; it is discussed in Chapter 18, "Remote Access to Hosts", with the rest of the Berkeley "r" commands. A more secure rcp replacement, scp, is also discussed in that chapter along with SSH, which scp is based on.

HTTP is actually also a file transfer protocol and can be used to transfer arbitrary files.

rsync, discussed in Chapter 22, "Administrative Services", is normally used for synchronization but may also be used as a generalized file transfer protocol over wide area networks.

icrosoft's NetMeeting, discussed in Chapter 19, "Real-Time Conferencing Services", also supports file transfer.
