17.4. Summary

NiftyTelnet SSH is a fine choice for a Macintosh SSH client. It supports scp, a rarity among SSH-1 implementations for PCs and Macs, and it conveniently remembers your passphrase until the program exits. On the other hand, NiftyTelnet SSH has the fewest features of any SSH implementation we've seen and can't generate key pairs. If you need more bells and whistles, such as port forwarding and a wider set of options, consider F-Secure SSH Client for the Mac (Chapter 16, "F-Secure SSH Client (Windows, Macintosh)").
